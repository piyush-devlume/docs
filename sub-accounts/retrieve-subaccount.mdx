Use this endpoint to fetch all details for a specific sub-account identified by its ID. The response includes authentication credentials, permissions, rate limits, and timestamps. No query parameters are required.

**GET** `https://api.vobiz.ai/api/v1/accounts/{auth_id}/sub-accounts/{sub_auth_id}`

## Path parameters

<ParamField path="auth_id" type="string" required>
  Your master account Auth ID (e.g., `MA_XXXXXX`).
</ParamField>

<ParamField path="sub_auth_id" type="string" required>
  The unique identifier of the sub-account to retrieve (e.g., `SA_67401KW8`).
</ParamField>

## Request examples

<CodeGroup>

```bash cURL
curl -X GET 'https://api.vobiz.ai/api/v1/accounts/{auth_id}/sub-accounts/{sub_auth_id}' \
  -H 'X-Auth-ID: {auth_id}' \
  -H 'X-Auth-Token: {auth_token}'
```

```python Python
import requests

auth_id = "YOUR_AUTH_ID"
sub_auth_id = "SA_67401KW8"
url = f"https://api.vobiz.ai/api/v1/accounts/{auth_id}/sub-accounts/{sub_auth_id}"
headers = {
    "X-Auth-ID": auth_id,
    "X-Auth-Token": "YOUR_AUTH_TOKEN"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```javascript Node.js
const axios = require("axios");

const authId = "YOUR_AUTH_ID";
const subAuthId = "SA_67401KW8";

const response = await axios.get(
  `https://api.vobiz.ai/api/v1/accounts/${authId}/sub-accounts/${subAuthId}`,
  {
    headers: {
      "X-Auth-ID": authId,
      "X-Auth-Token": "YOUR_AUTH_TOKEN",
    },
  }
);

console.log(response.data);
```

</CodeGroup>

## Response fields

<ResponseField name="id" type="string">
  Unique identifier for the sub-account (e.g., `SA_67401KW8`).
</ResponseField>

<ResponseField name="name" type="string">
  Name of the sub-account.
</ResponseField>

<ResponseField name="email" type="string">
  Email address associated with the sub-account.
</ResponseField>

<ResponseField name="phone" type="string">
  Phone number associated with the sub-account.
</ResponseField>

<ResponseField name="description" type="string">
  Description of the sub-account purpose or usage.
</ResponseField>

<ResponseField name="permissions" type="object">
  Permissions granted to this sub-account.

  <Expandable title="permissions fields">
    <ResponseField name="calls" type="boolean">
      Whether the sub-account can make and receive calls.
    </ResponseField>
    <ResponseField name="cdr" type="boolean">
      Whether the sub-account can access Call Detail Records.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="rate_limit" type="integer">
  API rate limit (requests per time period).
</ResponseField>

<ResponseField name="parent_auth_id" type="string">
  Auth ID of the parent master account.
</ResponseField>

<ResponseField name="auth_id" type="string">
  Authentication ID for this sub-account.
</ResponseField>

<ResponseField name="is_active" type="boolean">
  Whether the sub-account is currently active and can make API calls.
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of creation.
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of last update.
</ResponseField>

<ResponseField name="last_used" type="string">
  ISO 8601 timestamp of the last API call. `null` if never used.
</ResponseField>

<ResponseField name="resource_uri" type="string">
  API resource URI for this sub-account.
</ResponseField>
