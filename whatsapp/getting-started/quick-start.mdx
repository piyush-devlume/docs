This guide walks you through every step needed to connect a WhatsApp Business channel to Vobiz and send your first message — both from the console and via the API. Before you begin, make sure you have a [Vobiz account](https://console.vobiz.ai), an active Meta Business Account, and a WhatsApp Business Account (WABA) set up in Facebook Business Manager. If you haven't done that yet, review the [prerequisites](/whatsapp/getting-started/prerequisites) first.

<Info>
  The entire setup takes about **five minutes** once you have all prerequisites ready.
</Info>

## What you'll learn

- How to connect a WhatsApp Business channel to Vobiz
- How to verify your phone number via OTP
- How to send a message from the console
- How to send a message via the API
- How to receive and respond to incoming messages

---

<Steps>
  <Step title="Connect your WhatsApp channel">
    Log in to the [Vobiz Console](https://console.vobiz.ai) and navigate to **Messaging → Channels** in the left sidebar. Click **Connect Channel**.

    On the connection method screen, select **Bring Your Own Number (BYON)**. This is the recommended method when you already have a Meta Business Account and WABA.

    Fill in the following fields using values from your Meta Business Manager:

    | Field | Where to find it |
    | --- | --- |
    | **WABA ID** | WhatsApp Manager → Settings → Business Info |
    | **Phone Number ID** | WhatsApp Manager → Phone Numbers → click your number |
    | **Access Token** | Meta Business Settings → System Users → Generate Token |
    | **Display Name** | How your business name appears to customers |

    <Note>
      Need detailed help retrieving these credentials? The [BYON setup guide](/whatsapp/channels/byon) provides step-by-step instructions with screenshots for every field.
    </Note>

    Click **Connect Channel** to submit. Vobiz verifies your credentials and proceeds to phone number verification.
  </Step>

  <Step title="Verify your phone number">
    After submitting your credentials, Vobiz prompts you to verify the phone number via OTP. Choose your preferred method — **SMS** or **Voice call** — and enter the six-digit code you receive.

    Click **Verify**. Once the code is accepted, your channel appears in the Channels list with a **Connected** status.

    <Warning>
      The phone number cannot currently be registered with the WhatsApp Business app or WhatsApp Messenger. If it is, deregister it from those apps before proceeding.
    </Warning>
  </Step>

  <Step title="Send your first message from the console">
    Go to **Messaging → Inbox**. Click **New Message**, enter the recipient's phone number in E.164 format (e.g., `+1234567890`), type your message, and press **Send** or hit Enter.

    You can also attach images, videos, documents, and audio by clicking the attachment icon.

    <Tip>
      You can only send free-form messages to contacts who have messaged you within the last 24 hours. To reach a contact outside that window, use a pre-approved [template message](/whatsapp/messaging#template-messages).
    </Tip>
  </Step>

  <Step title="Send your first message via the API">
    Use the Send Message API to send messages programmatically from your application. Replace the placeholder values with your real credentials and channel ID.

    <CodeGroup>

    ```bash cURL
    curl -X POST https://api.vobiz.ai/v1/messaging/send \
      -H "X-Auth-ID: YOUR_AUTH_ID" \
      -H "X-Auth-Token: YOUR_AUTH_TOKEN" \
      -H "Content-Type: application/json" \
      -d '{
        "channel_id": "ch_abc123",
        "to": "+1234567890",
        "type": "text",
        "text": {
          "body": "Hello! This is my first WhatsApp message from Vobiz."
        }
      }'
    ```

    ```python Python
    import requests

    response = requests.post(
        "https://api.vobiz.ai/v1/messaging/send",
        headers={
            "X-Auth-ID": "YOUR_AUTH_ID",
            "X-Auth-Token": "YOUR_AUTH_TOKEN",
            "Content-Type": "application/json",
        },
        json={
            "channel_id": "ch_abc123",
            "to": "+1234567890",
            "type": "text",
            "text": {
                "body": "Hello! This is my first WhatsApp message from Vobiz."
            },
        },
    )
    print(response.json())
    ```

    ```javascript Node.js
    const response = await fetch("https://api.vobiz.ai/v1/messaging/send", {
      method: "POST",
      headers: {
        "X-Auth-ID": "YOUR_AUTH_ID",
        "X-Auth-Token": "YOUR_AUTH_TOKEN",
        "Content-Type": "application/json",
      },
      body: JSON.stringify({
        channel_id: "ch_abc123",
        to: "+1234567890",
        type: "text",
        text: {
          body: "Hello! This is my first WhatsApp message from Vobiz.",
        },
      }),
    });
    const data = await response.json();
    console.log(data);
    ```

    </CodeGroup>

    A successful request returns HTTP 200 with the following body:

    ```json
    {
      "success": true,
      "data": {
        "message_id": "wamid.HBgNMTIzNDU2Nzg5MAA=",
        "status": "sent",
        "to": "+1234567890",
        "timestamp": "2026-03-19T10:30:00Z"
      }
    }
    ```

    - `YOUR_AUTH_ID` — find this in Vobiz Console under **Settings → API Keys**
    - `YOUR_AUTH_TOKEN` — find this in the same location; keep it secret
    - `ch_abc123` — the channel ID shown on your Channels page after connecting
  </Step>

  <Step title="Receive and respond to incoming messages">
    When a customer messages your number, the conversation appears in **Messaging → Inbox** in real time. The inbox updates instantly via WebSocket — no need to refresh.

    Click any conversation to open the message thread, then type your reply in the text box at the bottom and press **Enter** to send.

    To receive message events in your own application, set up a webhook in the next step.
  </Step>

  <Step title="Set up webhooks (optional)">
    Webhooks let your server receive real-time notifications when messages arrive, are delivered, or are read.

    1. Go to **Messaging → Webhooks** and click **Create Webhook**.
    2. Enter a publicly accessible HTTPS URL for your endpoint.
    3. Select the events you want to subscribe to (e.g., `message.received`, `message.delivered`).
    4. Copy the signing secret displayed — use it to verify incoming webhook requests.
    5. Click **Save**.

    Here is an example `message.received` payload your endpoint will receive:

    ```json
    {
      "event": "message.received",
      "event_id": "evt_msg_recv_123",
      "timestamp": "2026-03-19T10:35:00Z",
      "channel_id": "ch_abc123",
      "data": {
        "message_id": "wamid.HBgNMTIzNDU2Nzg5MAA=",
        "from": "+1234567890",
        "type": "text",
        "text": {
          "body": "Hi, I need help with my order"
        },
        "timestamp": "2026-03-19T10:35:00Z"
      }
    }
    ```

    See the [Webhooks guide](/whatsapp/webhooks) for full details on signature verification and handling all event types.
  </Step>
</Steps>
