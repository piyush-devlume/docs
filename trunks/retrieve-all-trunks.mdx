Use this endpoint to retrieve a paginated list of all SIP trunks in your account. The response includes a `meta` object with pagination state and an `objects` array containing full trunk records.

## Request

```
GET https://api.vobiz.ai/api/v1/account/{auth_id}/trunks
```

**Headers**

| Header | Value |
|--------|-------|
| `X-Auth-ID` | Your account Auth ID |
| `X-Auth-Token` | Your account Auth Token |

## Path parameters

<ParamField path="auth_id" type="string" required>
  Your Vobiz account identifier.
</ParamField>

## Query parameters

<ParamField query="limit" type="integer">
  Number of trunks to return per page. Default: `20`. Maximum: `100`.
</ParamField>

<ParamField query="offset" type="integer">
  Number of trunks to skip before returning results. Default: `0`. Use with `limit` for page-based navigation.
</ParamField>

## Response

<ResponseField name="meta" type="object">
  Pagination metadata.

  <Expandable title="meta fields">
    <ResponseField name="limit" type="integer">
      Number of results returned in this page.
    </ResponseField>
    <ResponseField name="offset" type="integer">
      Number of results skipped.
    </ResponseField>
    <ResponseField name="total_count" type="integer">
      Total number of trunks in your account. Use this to calculate total pages.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="objects" type="array">
  Array of trunk objects. Each item has the same structure as the [trunk object](/trunks/trunk-object).
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl -X GET "https://api.vobiz.ai/api/v1/account/{Auth_ID}/trunks?limit=20&offset=0" \
  -H "X-Auth-ID: YOUR_AUTH_ID" \
  -H "X-Auth-Token: YOUR_AUTH_TOKEN"
```

```python Python
import requests

url = "https://api.vobiz.ai/api/v1/account/{Auth_ID}/trunks"
headers = {
    "X-Auth-ID": "YOUR_AUTH_ID",
    "X-Auth-Token": "YOUR_AUTH_TOKEN"
}
params = {"limit": 20, "offset": 0}

response = requests.get(url, headers=headers, params=params)
data = response.json()

print(f"Total trunks: {data['meta']['total_count']}")
for trunk in data["objects"]:
    print(trunk["trunk_id"], trunk["name"])
```

```javascript Node.js
const response = await fetch(
  "https://api.vobiz.ai/api/v1/account/{Auth_ID}/trunks?limit=20&offset=0",
  {
    headers: {
      "X-Auth-ID": "YOUR_AUTH_ID",
      "X-Auth-Token": "YOUR_AUTH_TOKEN"
    }
  }
);

const data = await response.json();
console.log(`Total trunks: ${data.meta.total_count}`);
data.objects.forEach(trunk => console.log(trunk.trunk_id, trunk.name));
```

<Tip>
To paginate, increment `offset` by `limit` for each subsequent page. For example, with `limit=20`: page 1 uses `offset=0`, page 2 uses `offset=20`, page 3 uses `offset=40`. Stop when `objects` is empty or when `offset >= total_count`.
</Tip>
