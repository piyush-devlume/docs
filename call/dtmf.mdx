DTMF (Dual-Tone Multi-Frequency) refers to the tones generated when a key is pressed on a phone keypad. The Vobiz DTMF API lets you send these tones programmatically during an active call, enabling fully automated interactions with IVR systems, conference bridges, voicemail systems, and any other tone-driven phone service.

## Base endpoint

```
https://api.vobiz.ai/api/v1/Account/{auth_id}/Call/{call_uuid}/DTMF/
```

## Supported digits

You can send any standard DTMF character:

- Numeric digits: `0` through `9`
- Star: `*`
- Hash: `#`

## Common use cases

- **IVR menu navigation** — Select menu options and sub-menus without human interaction.
- **Conference codes** — Join conference bridges by entering PINs automatically.
- **Automated survey responses** — Submit responses to phone-based surveys or feedback systems.
- **Account verification** — Enter account numbers or verification codes during calls.
- **Phone system control** — Send commands to systems that accept DTMF input.
