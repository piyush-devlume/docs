title: "Call Management: Voice API Overview"
description: "Manage outbound and inbound calls with Vobiz — initiate calls, transfer, record, play audio, send DTMF, and detect answering machines."
---

The Vobiz Call API gives you complete programmatic control over voice calls. A Call object is created whenever an outbound call is initiated or an inbound call is received. Use it to interact with live calls, retrieve call details, build dynamic call flows, and automate every step of the call lifecycle.

## Base URL

```
https://api.vobiz.ai/api
```

## Endpoint

```
https://api.vobiz.ai/api/v1/Account/{auth_id}/Call/
```

## Authentication

Every request requires two headers identifying your account.

| Header | Description |
|---|---|
| `X-Auth-ID` | Your Vobiz account Auth ID |
| `X-Auth-Token` | Your Vobiz account Auth Token |

```bash
X-Auth-ID: YOUR_AUTH_ID
X-Auth-Token: YOUR_AUTH_TOKEN
Content-Type: application/json
```

<Info>
  You can find your Auth ID and Auth Token in the Vobiz console under account settings.
</Info>
