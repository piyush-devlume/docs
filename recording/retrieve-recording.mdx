Use this endpoint to retrieve the full details of a specific recording using its `recording_id`. The response includes the direct download URL, call association, storage duration, billing information, and precise timing data. You can obtain a `recording_id` from the List All Recordings endpoint or from webhook notifications when recordings are created.

**GET** `https://api.vobiz.ai/api/v1/Account/{auth_id}/Recording/{recording_id}/`

## Path parameters

<ParamField path="auth_id" type="string" required>
  Your account Auth ID (e.g., `MA_XXXXXX`).
</ParamField>

<ParamField path="recording_id" type="string" required>
  The unique identifier of the recording to retrieve.
</ParamField>

## Request examples

<CodeGroup>

```bash cURL
curl -X GET https://api.vobiz.ai/api/v1/Account/{Auth_ID}/Recording/d7801b2e-e76d-4dd8-be9c-9e015a7267b8/ \
  -H "X-Auth-ID: YOUR_AUTH_ID" \
  -H "X-Auth-Token: YOUR_AUTH_TOKEN"
```

```python Python
import requests

auth_id = "YOUR_AUTH_ID"
recording_id = "d7801b2e-e76d-4dd8-be9c-9e015a7267b8"
url = f"https://api.vobiz.ai/api/v1/Account/{auth_id}/Recording/{recording_id}/"
headers = {
    "X-Auth-ID": auth_id,
    "X-Auth-Token": "YOUR_AUTH_TOKEN"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```javascript Node.js
const axios = require("axios");

const authId = "YOUR_AUTH_ID";
const recordingId = "d7801b2e-e76d-4dd8-be9c-9e015a7267b8";

const response = await axios.get(
  `https://api.vobiz.ai/api/v1/Account/${authId}/Recording/${recordingId}/`,
  {
    headers: {
      "X-Auth-ID": authId,
      "X-Auth-Token": "YOUR_AUTH_TOKEN",
    },
  }
);

console.log(response.data);
```

</CodeGroup>

## Response fields

The response wraps the recording in a standard list response envelope. See the [Recording object](/recording/recording-object) for complete field descriptions.

<ResponseField name="api_id" type="string">
  Unique identifier for this API request. Use this for support or debugging.
</ResponseField>

<ResponseField name="meta" type="object">
  Pagination metadata including `limit`, `offset`, and `total_count`.
</ResponseField>

<ResponseField name="objects" type="array">
  Array containing the single matching recording object.
</ResponseField>
