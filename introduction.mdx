Vobiz is a developer platform for voice and telephony. It gives you enterprise-grade SIP trunking, a programmable Voice XML API, and direct integrations with leading AI voice agent platforms — all accessible through a single REST API and console. Whether you are building an AI-powered call center, a custom IVR, or a real-time voice bot, Vobiz provides the carrier connectivity and infrastructure layer your application needs.

## Authentication

All API requests require two headers:

| Header | Description |
|---|---|
| `X-Auth-ID` | Your account Auth ID |
| `X-Auth-Token` | Your account Auth Token |

Find both values in the **Account Settings** section of the [Vobiz console](https://console.vobiz.ai).

```bash
curl -X GET "https://api.vobiz.ai/api/v1/account/{auth_id}/..." \
  --header "X-Auth-ID: {auth_id}" \
  --header "X-Auth-Token: {auth_token}"
```
