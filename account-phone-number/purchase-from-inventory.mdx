Use this endpoint to purchase a phone number from Vobiz inventory and assign it to your account. Before calling this endpoint, use the list inventory endpoint to confirm the number is available. The operation immediately debits your account balance for the setup fee plus the monthly fee. For sub-accounts (`SA_`), the charge is applied to the parent master account.

**POST** `https://api.vobiz.ai/api/v1/account/{auth_id}/numbers/purchase-from-inventory`

<Warning>
  If your account has insufficient balance, the purchase fails and the number remains in inventory. Verify your balance before purchasing.
</Warning>

## Path parameters

<ParamField path="auth_id" type="string" required>
  Your account Auth ID (e.g., `MA_XXXXXX`).
</ParamField>

## Body parameters

<ParamField body="e164" type="string" required>
  Phone number to purchase in E.164 format (e.g., `+919876543210`).
</ParamField>

<ParamField body="currency" type="string">
  Currency for the transaction. Defaults to the number's native currency or `USD` if not specified.
</ParamField>

## Request examples

<CodeGroup>

```bash cURL
curl -X POST https://api.vobiz.ai/api/v1/account/{AUTH_ID}/numbers/purchase-from-inventory \
  -H "X-Auth-ID: {AUTH_ID}" \
  -H "X-Auth-Token: {AUTH_TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "e164": "+919876543210",
    "currency": "USD"
  }'
```

```python Python
import requests

auth_id = "YOUR_AUTH_ID"
url = f"https://api.vobiz.ai/api/v1/account/{auth_id}/numbers/purchase-from-inventory"
headers = {
    "X-Auth-ID": auth_id,
    "X-Auth-Token": "YOUR_AUTH_TOKEN",
    "Content-Type": "application/json"
}
payload = {
    "e164": "+919876543210",
    "currency": "USD"
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
```

```javascript Node.js
const axios = require("axios");

const authId = "YOUR_AUTH_ID";
const response = await axios.post(
  `https://api.vobiz.ai/api/v1/account/${authId}/numbers/purchase-from-inventory`,
  {
    e164: "+919876543210",
    currency: "USD",
  },
  {
    headers: {
      "X-Auth-ID": authId,
      "X-Auth-Token": "YOUR_AUTH_TOKEN",
      "Content-Type": "application/json",
    },
  }
);

console.log(response.data);
```

</CodeGroup>

## Response fields

<ResponseField name="message" type="string">
  Confirmation message on success (e.g., `purchased from inventory`).
</ResponseField>

<ResponseField name="number" type="object">
  The updated PhoneNumber object with your `auth_id` now assigned.

  <Expandable title="PhoneNumber fields">
    <ResponseField name="id" type="string">
      UUID identifier for the phone number.
    </ResponseField>
    <ResponseField name="auth_id" type="string">
      Your account Auth ID, now assigned to this number.
    </ResponseField>
    <ResponseField name="e164" type="string">
      Phone number in E.164 format.
    </ResponseField>
    <ResponseField name="country" type="string">
      Country code (e.g., `IN`, `US`).
    </ResponseField>
    <ResponseField name="region" type="string">
      State or region code.
    </ResponseField>
    <ResponseField name="status" type="string">
      Number status. Will be `active` after a successful purchase.
    </ResponseField>
    <ResponseField name="setup_fee" type="number">
      One-time setup fee that was charged.
    </ResponseField>
    <ResponseField name="monthly_fee" type="number">
      Monthly recurring fee for the number.
    </ResponseField>
    <ResponseField name="currency" type="string">
      Currency code for the fees.
    </ResponseField>
    <ResponseField name="purchased_at" type="string">
      ISO 8601 timestamp when the purchase was completed.
    </ResponseField>
  </Expandable>
</ResponseField>
